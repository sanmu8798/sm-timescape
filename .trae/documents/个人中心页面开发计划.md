# 个人中心页面开发计划

## 背景

当前 `PageProfile.vue` 个人中心页面存在大量仅展示、无交互的入口：
- 「更多服务」中的账号与安全、隐私设置、帮助与反馈、关于时光集均未绑定点击事件
- 「我的内容」中的收藏夹、我的动态缺少对应页面
- 顶部用户资料卡无编辑入口
- 缺少退出登录入口

本次计划补齐上述缺失，同时保证 mock 数据与项目现有数据模型一致。

## 目标

1. 为所有未开发功能入口统一提供「功能尚未开发，敬请期待」Toast 提示
2. 新增「编辑个人资料」页面，支持修改昵称、个性签名、手机号，并同步更新 Store 与 mock 数据
3. 新增「我的收藏」页面，展示示例收藏数据
4. 新增「我的动态」页面，展示当前用户发布的打卡动态
5. 新增「退出登录」入口，点击提示功能尚未开发
6. 注册新路由，保持与现有路由风格一致

## 详细方案

### 1. PageProfile.vue 入口梳理

保持现有视觉结构，为以下入口补充点击行为：

- 顶部资料卡（含箭头）-> `PageEditProfile`
- 数据看板：打卡记录 -> `PageCheckInRecord`（已有）、获得徽章 -> `PageBadges`（已有）
- 我的内容：收藏夹 -> `PageFavorites`、我的动态 -> `PageMyPosts`
- 探索记录：点亮进度 -> `onAlert`、成就徽章 -> `PageBadges`、足迹时间线 -> `PageCheckInRecord`
- 更多服务：账号与安全 / 隐私设置 / 帮助与反馈 / 关于时光集 -> `onAlert`
- 页面底部新增「退出登录」按钮 -> `onAlert`

统一使用与 `PageIndex.vue` 相同的 `onAlert` 提示函数：
```js
const onAlert = () => showToast({ message: "功能尚未开发，敬请期待", icon: "warning-o" })
```

### 2. 编辑个人资料页（PageEditProfile.vue）

布局参考 `PageCheckIn.vue` 的表单页风格：
- 顶部返回栏 + 标题 + 保存按钮
- 头像展示区（仅展示，暂不支持上传）
- 表单项：昵称、个性签名、手机号
- 保存时调用 `updateProfile({ nickname, bio, phone })`，更新 `mock/user.js` 中的 `profile` 对象
- 同步调用 `baseStore.setUser(profile)` 刷新全局用户数据
- 返回个人中心

### 3. 我的收藏页（PageFavorites.vue）

布局参考 `PageCheckInRecord.vue` 的列表页风格：
- 顶部返回栏 + 标题
- 收藏列表卡片：图片、名称、类型标签、所属城市
- 数据来源：`src/mock/favorites.js`，数据结构保持与 `destinations.js` / `checkins.js` 的 `city_id`、`target_type`、`target_id` 一致

示例收藏数据：
- 城市：北京
- 景点：故宫博物院
- 美食：重庆火锅
- 风俗：川剧变脸

### 4. 我的动态页（PageMyPosts.vue）

直接使用 `getCheckinsByUser` 获取当前用户动态，复用 `PageCheckInRecord.vue` 的时间线样式：
- 顶部返回栏 + 标题
- 按时间倒序展示当前用户的打卡记录
- 支持点击进入对应目标详情页

### 5. Mock 数据更新

- 新增 `updateProfile(data)` 方法，合并字段到 `profile`

### 6. 路由注册

在 `src/router/index.js` 新增：
```js
import PageEditProfile from "@/views/PageEditProfile.vue"
import PageFavorites from "@/views/PageFavorites.vue"
import PageMyPosts from "@/views/PageMyPosts.vue"

{ path: "/profile/edit", name: "PageEditProfile", component: PageEditProfile, meta: { title: "编辑资料" } },
{ path: "/profile/favorites", name: "PageFavorites", component: PageFavorites, meta: { title: "我的收藏" } },
{ path: "/profile/posts", name: "PageMyPosts", component: PageMyPosts, meta: { title: "我的动态" } },
```

## 验证方式

1. 运行 `npm run dev`
2. 进入「我的」页面
3. 点击顶部资料卡 -> 进入编辑资料页，修改昵称后保存，返回查看是否更新
4. 点击「收藏夹」-> 进入收藏列表，检查图片类型是否正确
5. 点击「我的动态」-> 进入动态列表，检查是否只展示当前用户动态
6. 点击「账号与安全」等菜单项 -> 弹出 Toast 提示
7. 点击「退出登录」-> 弹出 Toast 提示
